# Render one report per sample

Render one report per sample

## Usage

``` r
render_report_batch(x, output_dir = "reports", overwrite = FALSE)
```

## Arguments

- x:

  An `aegis` or `aegis_multi` object.

- output_dir:

  Output directory.

- overwrite:

  Whether to overwrite existing files.

## Value

List with report paths and batch summary path.
