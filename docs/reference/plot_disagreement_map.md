# Plot spot-level consensus disagreement map

Plot spot-level consensus disagreement map

## Usage

``` r
plot_disagreement_map(x, palette = "nature", base_size = 12)
```

## Arguments

- x:

  An `aegis` object.

- palette:

  Palette family: `nature`, `viridis`, `scico`, or `brewer`.

- base_size:

  Base font size.

## Value

A plot object (Seurat spatial plot when available, otherwise ggplot).
