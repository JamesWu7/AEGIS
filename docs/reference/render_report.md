# Render an AEGIS HTML report

Renders a standalone HTML report from the package template.

## Usage

``` r
render_report(
  x,
  output_file = "aegis_report.html",
  title = "AEGIS Deconvolution Audit Report"
)
```

## Arguments

- x:

  An `aegis` object.

- output_file:

  Output HTML path.

- title:

  Report title.

## Value

Path to rendered HTML report.
