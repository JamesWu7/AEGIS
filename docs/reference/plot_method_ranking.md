# Plot method ranking summary

Plot method ranking summary

## Usage

``` r
plot_method_ranking(x, sample = NULL, palette = "nature", base_size = 12)
```

## Arguments

- x:

  An `aegis` object.

- sample:

  Optional sample ID for multi-sample objects.

- palette:

  Palette family: `nature`, `viridis`, `scico`, or `brewer`.

- base_size:

  Base font size.

## Value

A ggplot object.
